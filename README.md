# fraudlens-hackathon
AI/ML dashboard for real-time mule account detection and  fraud transaction analysis. Submission for PSB's Cybersecurity,  Fraud &amp; AI Hackathon 2026 in collaboration with IIT Hyderabad,  powered by Bank of India &amp; Dept. of Financial Services, MoF.
